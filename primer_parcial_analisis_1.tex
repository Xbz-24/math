	\section{}
	Compute:
	\[
	\lim_{{n \to +\infty}} \frac{2^{-n} + 12n}{\sqrt{4n^2 + 5} + 2n}
	\]
	
	\section{}
	Let \( f: \mathbb{R} \to \mathbb{R} \) be defined as
	\[
	f(x) = 
	\begin{cases} 
		\frac{\cos(6x) - 1}{\ln(6x+1)} & \text{if } x > 0 \\
		-3x & \text{if } x \leq 0 
	\end{cases}
	\]
	Determine if \( f \) is differentiable at \( x = 0 \), and if it is, find \( f'(0) \).
